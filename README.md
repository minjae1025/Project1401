# Mirim-Lotto
2024 Java 수행평가이자, 첫 JAVA 프로젝트입니다. 단순 시뮬레이션이 아닌, 로또와 미니게임을 통해 즐기는 게임입니다.

## 게임 설명
1. 가상의 돈을 이용해 로또를 구매할 수 있습니다
2. 돈이 부족하다면 알바(미니게임)을 이용해 돈을 벌 수 있습니다.
3. 게임 저장을 하시면 나중에 이어서 할 수 있습니다.

## 실행 조건
- 윈도우 10 이상의 윈도우 PC
- JAVA 17 버전 이상
- Eclipse 2023.12 버전 이상
